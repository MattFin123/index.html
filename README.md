# NutriFlow

Your personal maingain fitness & nutrition tracker. A fully offline-capable web app — no account, no server, no subscription.

## How to use on your phone

1. Go to `https://YOUR-USERNAME.github.io/fuellift/`
2. In Safari (iPhone) or Chrome (Android), tap **Share → Add to Home Screen**
3. Done — it installs like a native app

## Features

- 📊 Daily calorie & macro tracking (protein, carbs, fat)
- 🔍 Food search via Open Food Facts database
- 📷 Barcode scanner for packaged foods
- ⚖️ Weight tracker with BMI and 7-day chart
- 🏋️ Workout logging with sets, reps & weight
- ⏱️ Rest timer between sets
- 🏆 Rank system (Wood → Bronze → Silver → Gold → Platinum → Diamond → Champion → Titan → Olympian) based on your bench press relative to bodyweight
- 🥇 Trophy shelf for achievements
- 💪 Exercise PR tracking and strength calculator
- 💡 Food tips and smart macro alerts
- 💾 All data saved locally on your device

## How to deploy on GitHub Pages

1. Create a new GitHub repo called `fuellift`
2. Upload `index.html` (rename FuelLift.html to index.html)
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Save — your app will be live at `https://YOUR-USERNAME.github.io/fuellift/`
